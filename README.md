## Installation
#### Requirements:

- <strong>PHP 7.4
- Node
- MySQL</strong>

#### Configuration:

- Open the .env file and configure the APP_KEY and APP_DEBUG

```
APP_KEY= "YOUR GENERATED APP KEY"
APP_DEBUG=true
```
> to generate APP_KEY run this command:
```
php artisan key:generate
```

- Then run

```
php artisan serve
```
